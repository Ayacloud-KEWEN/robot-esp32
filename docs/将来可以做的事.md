# 将来可以做的事 · 创意清单

> 基于你已有的资源盘点出的"能做且有趣"的事。难度：⭐ 入门 → ⭐⭐⭐⭐⭐ 大工程。
> 你的家底：小智服务器(树莓派) + ePaper-1.54 套件 + S3 机芯盒 + DeepSeek + Home Assistant + WiZ 智能灯 + go2rtc 摄像头 + 1TB 媒体库(Jellyfin/Navidrome/电子书) + Tailscale + 公网域名 robot.ayacloud.fr

## A. 人格与灵魂（纯软件，最快见效）

| 想法 | 难度 | 说明 |
|---|---|---|
| **专属人格定制** | ⭐ | 改提示词塑造独一无二的性格：口头禅、说话节奏、幽默感、价值观。可以做多个角色一键切换（管家/损友/温柔姐姐/东北大爷） |
| **专属声音** | ⭐⭐ | 换 TTS 音色；进阶用 CosyVoice 等克隆一个声音（自己的声音/家人的声音，用自己声音做机器人格外有趣） |
| **长期记忆** | ⭐⭐ | 开启 Memory 模块，让它记住你的生日、喜好、上次聊的话题——"陪伴感"的核心来源 |
| **声纹识别分人** | ⭐⭐ | 家里每个人说话它都认得出，对你喊"船长"，对孩子讲故事换语气 |
| **私人知识库(RAG)** | ⭐⭐⭐ | 对接 RAGFlow，把说明书、家庭档案、你写的文档喂给它，问啥都答得上 |

## B. 让它有"手脚"——智能家居联动（家底现成，性价比最高 ★推荐先做）

| 想法 | 难度 | 说明 |
|---|---|---|
| **语音控制 WiZ 灯** | ⭐⭐ | 你已有自写的 wiz-control 服务！写个插件对接，"来点看电影的氛围" → 灯光变暗变暖 |
| **Home Assistant 全屋控制** | ⭐⭐ | 首选走 MCP 通道：`.mcp_server_settings.json` 接 HA 的 MCP Server，一次接入全屋设备（步骤见学习文档08篇第9节）；hass 插件为备选方案 |
| **摄像头之眼** | ⭐⭐⭐ | go2rtc 已在跑：问"门口有人吗？"→ 抓一帧图 → DeepSeek 视觉模型分析 → 语音告诉你 |
| **早安播报** | ⭐⭐ | 定时任务：早上被唤醒时播报天气、日程、新闻——用你自己定制的声音 |
| **提醒管家** | ⭐⭐ | "提醒我下午三点开会"、吃药提醒、倒垃圾提醒（写一个提醒插件+定时触发） |

## C. 媒体与内容（1TB 硬盘派上用场）

| 想法 | 难度 | 说明 |
|---|---|---|

| **讲故事/读电子书** | ⭐⭐⭐ | `/mnt/ssd/ebook` 里的书 → 让它接着上次的进度读给你听，睡前故事神器 |

## D. 硬件形态进化（ePaper 套件 + S3 机芯盒登场）

| 想法 | 难度 | 说明 |
|---|---|---|
| **ePaper 桌面伴侣** | ⭐⭐ | 烧录 ePaper-1.54 板连自建服务器——第一台实体机器人！墨水屏常显时钟/天气/今日一句，说话时显示对话 |
| **自定义唤醒词** | ⭐⭐ | 不叫"小智"，给它起个专属名字 |
| **定制表情包** | ⭐⭐ | 用 assets-generator 自制墨水屏表情（像素风/手绘风特别配墨水屏） |
| **机芯盒装进"身体"** | ⭐⭐⭐ | 给 S3 机芯盒设计外壳（3D 打印/木盒/毛绒玩具肚子里），做真正的陪伴机器人本体 |
| **会动的机器人** | ⭐⭐⭐⭐ | 加舵机：点头、摇头、转身，注册成 MCP 工具让 LLM 决定什么时候做动作（"同意就点点头"） |
| **传感器感知** | ⭐⭐⭐ | 温湿度/光线/人体感应：有人走近主动打招呼；房间太热主动提议开空调 |
| **电池化+便携** | ⭐⭐⭐ | 锂电池+充电管理，抱着走的机器人；出门自动切公网 robot.ayacloud.fr |

## E. 大工程（技能攒够后的 Boss 关）

| 想法 | 难度 | 说明 |
|---|---|---|
| **UI 全面魔改** | ⭐⭐⭐ | 智控台登录页/管理界面换成你的品牌风格（Vue + Element UI），做成"Ayacloud 机器人控制中心" |
